# Netflix Data Analysis

## Project Overview

This project analyzes Netflix Movies and TV Shows data using Python and data visualization techniques.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Insights

- Movies are more numerous than TV Shows.
- International Movies is the most common genre.
- The United States contributes the highest number of titles.
- TV-MA is the most common rating.
- Average movie duration is approximately 99.5 minutes.
- Netflix added the highest amount of content in 2019.

## Conclusion

This project demonstrates data cleaning, exploratory data analysis, and visualization skills using a real-world dataset.
